# 📊 Electronic E-Commerce Customer Behavior Analysis

## 📌 Project Overview
This project analyzes customer behavior in an electronics e-commerce platform using event-level data (view, cart, purchase). The analysis focuses on understanding conversion performance, retention patterns, customer value distribution, and revenue concentration.

---

## 🎯 Objectives
- Evaluate user conversion funnel performance
- Analyze category and brand performance
- Analyze customer retention and repeat purchase behavior  
- Identify high-value customer segments using RFM  
- Generate actionable business recommendations  

---

## 🧠 Methods Used
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)  
- Funnel Analysis (View → Cart → Purchase)  
- Cohort Analysis (User Retention & Revenue Retention)  
- RFM Analysis
- Customer Segmentation (K-Means Clustering)  

---

## 🛠 Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib / Seaborn / Plotly  
- Scikit-learn  

---

## 📊 Key Insights

- Significant drop-off occurs between **View → Cart**, indicating weak purchase intent.
- Customer retention drops sharply after the first month.
- Majority of customers are one-time buyers.
- Revenue is highly concentrated among a small segment of high-value customers.
- Frequency and Monetary distributions are heavily right-skewed.

---

## 💡 Business Recommendations

- Optimize product pages and pricing strategy to improve View → Cart conversion.
- Implement structured retention campaigns (email remarketing, post-purchase engagement).
- Develop loyalty programs targeting high-value customers.
- Focus on increasing repeat purchase rate rather than relying solely on new customer acquisition.

---

## 🏁 Conclusion

The business demonstrates strong acquisition but weak retention.  
Long-term growth should prioritize improving conversion efficiency and customer lifecycle management to build sustainable revenue.
